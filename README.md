# Linear Regression
This project demonstrates 3 regression techniques applied to a Netflix dataset:
1. 📉 Linear Regression
2. 🌲 Random Forest Regressor
3. 🔁 MultiOutput Regressor
## 📊 Target Predictions:
- Calculated Fuel
- Fuel of Disb
- Total Fuel
- Paid Days Fuel
## 🔍 Features Used:
- Loans in Demand
- Disb Count
- Working Day
## Each model includes:
- Train/test split
- Model training
- Evaluation metrics: MAE, MSE, RMSE, R²
- Prediction from user input
